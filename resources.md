# Improve furniture render accuracy, styling consistency,

# CORE SYSTEM RULES

The AI must preserve:
- Exact furniture silhouette
- Exact proportions
- Exact stitching direction
- Exact armrest geometry
- Exact leg structure
- Exact curvature
- Exact upholstery segmentation
- Exact material hierarchy

DO NOT:
- Invent random shapes
- Over-modernize classic luxury pieces
- Convert luxury Italian furniture into generic minimalist furniture
- Change chair ergonomics
- Add fake tufting
- Add fake piping
- Add random Scandinavian styling unless specified

The system should prioritize:
- Italian luxury styling
- Contemporary luxury interiors
- High-end hospitality aesthetics
- Milan showroom styling
- Soft dramatic lighting
- Premium material realism
